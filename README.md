# Rayvise

High-end tile art portfolio website featuring dark theme and cinematic animations.

## 🌐 Live Site

**https://rayvise.ca**

## 🎨 Features

- React 19 + TypeScript
- Vite build system
- GSAP ScrollTrigger animations
- 3D card transforms with mouse tracking
- Custom cursor (desktop)
- Canvas particle field background
- Fully responsive design
- Reduced motion support

## 🛠️ Tech Stack

- **Framework:** React 19
- **Build Tool:** Vite 7
- **Styling:** Tailwind CSS 3 + tailwindcss-animate
- **Animations:** GSAP 3 (ScrollTrigger)
- **Icons:** Lucide React
- **Fonts:** Geist (Google Fonts)

## 📦 Development

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## 🚀 Deployment

Deployed to GitHub Pages via `gh-pages` branch.

Custom domain configured via DNS A records and CNAME.
