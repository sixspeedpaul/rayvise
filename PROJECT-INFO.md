## Overview

High-end photographer portfolio template with dark theme and cinematic animations.

## Features

- GSAP ScrollTrigger animations throughout
- 3D card transforms with mouse tracking
- Custom cursor (desktop only)
- Canvas particle field background
- Noise texture overlay
- Responsive design with mobile menu
- Reduced motion support

## Tech Details

- React 19 + TypeScript
- Vite 7
- Tailwind CSS 3 + tailwindcss-animate
- GSAP 3 (ScrollTrigger)
- Lucide React icons
- Geist font (Google Fonts)

## Content Configuration

All content managed in `src/config.ts`. Each section has its own config object.
